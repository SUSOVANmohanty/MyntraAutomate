Myntra Test Automation

Prerequisites:
Make sure you have Java installed.
Set up Selenium WebDriver.
Download ChromeDriver compatible with your Chrome version.

Project Structure:

|-- src
|   |-- test
|       |-- java
|           |-- com
|               |-- yourpackage
|                   |-- MyntraTest.java
|-- chromedriver.exe
|-- selenium-server-standalone.jar
|-- README.md

## Instructions

1. Clone the repository.
2. Download the appropriate `chromedriver.exe` for your Chrome version and place it in the project root.
3. Run the `selenium-server-standalone.jar` (if required).
4. Open `src/test/java/com/yourpackage/MyntraTest.java` and update the placeholders with your actual data.
5. Execute the `MyntraTest.java` file to run the automated test.

## Dependencies

- Selenium 4.17.0
- ChromeDriver 121.0.6167.139
- Chrome Browser

Feel free to reach out for any issues or questions.
